# sysOLA1EnterpriseIntegration
OLA1 i system integration

## Start here
- Report: `report/REPORT.md`

## Structure
- `report/` this folder contains the REPORT.md file, which is my report for the OLA hand-in. (https://github.com/VivekNagra/sysOLA1EnterpriseIntegration/blob/main/report/REPORT.md)
- `demo-pipes-filters/` Spring Boot demo implementing Pipes & Filters (run via Maven Wrapper: `.\mvnw.cmd spring-boot:run`). See the README in src/main/java/dk/vivek/pipesfiltersdemo for details.
- `diagrams/` I have used Mermaid in the report itself which you can check out in the /report folder.
